# WJXUpdateSet
    
## 介绍
    脚本设置app图标/启动图/版本号/bulid号

## 版本
    1.0.0

## 准备工作
### 修改脚本(低配版)
*  新建文件夹
*  放入updateSet.sh拖入要添加代码的项目
*  修改项目参数文件夹路径
 
   project_name=项目名字
    

## 使用方法
* 使用终端进入WJXUpdateSet文件
* 执行:sh updateSet.sh
